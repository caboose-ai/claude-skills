# Pre PR Review Loop

Run local review, fix findings, verify, open or update a PR, request Codex
review, loop through automated feedback, and leave the human as the final
reviewer.

## Install

```bash
npx skills add caboose-ai/ai-skills@pre-pr-review-loop -g
```

## Use When

- preparing to push or open a pull request
- updating a PR after local code changes
- marking a draft PR ready for review
- the user says "pr", "open pr", "ship", "push this", or "yeet"

## What It Produces

- local review findings and fixes
- Codex review findings and fixes
- validation commands and results
- remaining risks or manual checks
- explicit handoff to the human as final reviewer
