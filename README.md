# Claude Skills by Caboose AI

A collection of skills for Claude Code.

## Available Skills

### [maximum-effort-statusline](./maximum-effort-statusline/)

Powerline-style Claude Code status bar with dynamic segments (directory, model, cost, git) and a rotating quips easter egg when effort is set to max.

**Install:**

```bash
npx skills add caboose-ai/claude-skills@maximum-effort-statusline -g
```
